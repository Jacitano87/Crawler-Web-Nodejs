  <a href="/biomedicine/">
          Biomedicine        News  </a>
  </h2>
                    <ul style="display: none;">
          <li class="icon-comment">
            <a href="#comments">
              <span class="livefyre-commentcount" data-lf-site-id="296821" data-lf-article-id="nid_407222" data-lf-format-singular="@count comment" data-lf-format-plural="@count comments" data-lf-display-parents="ul">0 comments</span>            </a>

        <h1>A Safe and Simple Arsenic Detector</h1>
                  <p class="intro">
      Researchers have built a bacteria-based device that sniffs out the toxic chemical.  </p>
        
        
      
      
              <ul class="byline">
  <li>
                  By <a href="/contributor/jennifer-chu/">Jennifer Chu</a> on January 25, 2007                </li>
</ul>      
      
      
      
      
      <div class="clearfix"></div>

    </header>

    
    
    
    
    
    <div class="sidebar right">

                                <aside class="ad-right">
            <div id="dfp-ad-tr_www_body_rail_right_top_portrait-wrapper" class="dfp-tag-wrapper"><div  id="dfp-ad-tr_www_body_rail_right_top_portrait" class="dfp-tag-wrapper" data-cb-ad-id="Body rail right top portrait">
  <script type="text/javascript">
  googletag.cmd.push(function() {
    googletag.display("dfp-ad-tr_www_body_rail_right_top_portrait");
  });
  </script>
</div>
</div>          </aside>
              
      
      
      
    </div>

                <section class="body"><aside class="img-left">
  <img src="http://www.technologyreview.com/sites/default/files/legacy/arsenicbg_0.jpg" width="220" height="200" alt="" />  </aside>
    <aside class="caption-left">
            <p >A student team from the University of Edinburgh has genetically engineered bacteria to detect arsenic in water. In combination with a drop of pH indicator (far right), samples turn red (middle) in the presence of arsenic and yellow in its absence.</p>
    </aside>
  
<p>Researchers at the University of Edinburgh, in Scotland, have genetically modified the bacteria <i>E. coli</i> to detect trace amounts of arsenic in drinking water. They hope that the bacteria-based technology will eventually lead to safe, precise, and easy-to-use field arsenic test kits. Chris French, professor of microbial biotechnology at Edinburgh, says that such tests could be as easy to use as home pregnancy tests and would not require a trained technician. </p><p></p>  <aside class="ad-left">
              <div id="dfp-ad-tr_www_body_content_portrait_in_story-wrapper" class="dfp-tag-wrapper"><div  id="dfp-ad-tr_www_body_content_portrait_in_story" class="dfp-tag-wrapper" data-cb-ad-id="Body content portrait in story">
  <script type="text/javascript">
  googletag.cmd.push(function() {
    googletag.display("dfp-ad-tr_www_body_content_portrait_in_story");
  });
  </script>
</div>
</div>      </aside>
<p>Arsenic poisoning affects millions of people worldwide. One of the hardest hit countries is Bangladesh, where up to 35 million people get their drinking water from arsenic-contaminated wells, according to the World Health Organization. In recent years there has been a concerted effort to map the distribution of arsenic and flag contaminated wells. In some cases, scientists and aid workers have sent water samples to be tested in labs using fluorescence techniques–an expensive and time-consuming process. In other cases, they have used portable test kits. Most of these field tests, however, require training to operate and produce toxic chemicals, such as arsine gas. </p><p></p><p>“There’s definitely some drawbacks of existing field kits, and one is toxic disposal,” says Gregory Miller, an arsenic geochemist at Subsurface Technologies, an environmental remediation company in Socorro, NM. “An ideal detector should have a low detection limit, high precision, and no waste stream involved with it.”</p><p></p><p>The arsenic detector developed at Edinburgh is one of the first practical examples of a device built using the principles of synthetic biology. The aim of synthetic biology is, first, to identify the precise function of particular DNA sequences, or genetic parts. Some parts may control certain proteins. Others may act as switches for cellular processes. Then researchers mix and match to see how different parts fit together to form new genetic circuits, creating novel biological functions within living organisms, such as bacteria. </p><p></p><p>Drew Endy, assistant professor of biological engineering at MIT, likens synthetic biology to building with Lego. He has helped assemble a registry of standard genetic parts as a resource for scientists to borrow from, build on, and add to. Endy is also cofounder of iGEM, the annual International Genetically Engineered Machine competition, at which Edinburgh’s bacteria-based arsenic biosensor won the prize as best real-world application in 2006.</p><p></p><p>The Edinburgh group found that the bacterium <i>E. coli</i> possesses two seemingly unrelated genetic sequences that, in combination, form an effective arsenic-detection device. First, <i>E. coli</i> possesses a natural arsenic detoxification system that is switched on only in the presence of arsenic. The bacterium also naturally breaks down lactose to produce acid. The researchers isolated the arsenic-switch gene and attached it to the first gene involved in the breakdown of lactose. </p><p></p></section>

LF.CommentCount({
  replacer: function(element, count) {
    var formatSingular = element.getAttribute("data-lf-format-singular");
    var formatPlural = element.getAttribute("data-lf-format-plural");
    if (formatSingular && formatPlural) {
      if (count == 1) {
        element.innerHTML = formatSingular.replace("@count", count);
      } else {
        element.innerHTML = formatPlural.replace("@count", count);
      }
    } else {
      element.innerHTML = count;
    }

    var minComments = 1;
    // Get the minimum comments from the HTML attribute, if there is one.
    var htmlMinComments = element.getAttribute("data-lf-min-comments");
    if (htmlMinComments) {
      minComments = 1 * htmlMinComments;
    }

    var displayElement = element;
    // If we need to hide/show a parent element instead of the element itself,
    // we specify the jQuery .parent() selector in the attribute data-lf-display-parents.
    var displayParentSelector = element.getAttribute("data-lf-display-parents");
    if (displayParentSelector) {
      var displayElementParents = $(displayElement).parents(displayParentSelector);
      if (displayElementParents.length > 0)
        displayElement = displayElementParents[0];
    }

    var displayAltElement = null;
    var displayAltParentsSelector = element.getAttribute("data-lf-display-alt-parents");
    var displayAltSelector = element.getAttribute("data-lf-display-alt");
    if (displayAltParentsSelector && displayAltSelector) {
      displayAltParents = $(element).parents(displayAltParentsSelector);
      if (displayAltParents.length > 0) {
        var displayAltParent = displayAltParents[0];
        var displayAltElements = $(displayAltParent).find(displayAltSelector);
        if (displayAltElements.length > 0)
          displayAltElement = displayAltElements[0];
      }
    }

    // Hide the element if it doesn't have enough comments.
    if (count >= minComments) {
      displayElement.style.display = "";
      // Hide the alternate element.
      if (displayAltElement)
        displayAltElement.style.display = "none";
    } else {
      displayElement.style.display = "none";
      // Show the alternate element.
      if (displayAltElement)
        displayAltElement.style.display = "";
    }
  }
});
